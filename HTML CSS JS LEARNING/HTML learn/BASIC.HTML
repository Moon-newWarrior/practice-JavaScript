Document Structure: <html>: Defines the root of an HTML document. <head>: Contains meta-information about the HTML document. <title>: Sets the title of the HTML document (appears in the browser tab). <body>: Contains the content of the HTML document. Text Formatting: <p>: Defines a paragraph. <h1>, <h2>, ..., <h6>: Define heading levels. <strong> or <b>: Represents strong importance (bold). <em> or <i>: Represents emphasized text (italic). <u>: Represents underlined text. Lists: <ul>: Defines an unordered list. <ol>: Defines an ordered list. <li>: Defines a list item. Links and Navigation: <a>: Defines a hyperlink. <nav>: Defines a set of navigation links. Images: <img>: Embeds an image. Tables: <table>: Defines a table. <tr>: Defines a table row. <td>: Defines a table cell. Forms: <form>: Defines an HTML form. <input>: Defines an input field. <label>: Defines a label for an <input> element. <select>: Defines a dropdown list. <textarea>: Defines a multiline text input. Semantic Elements: <article>: Represents a self-contained piece of content. <section>: Represents a section of a document. <header>: Defines a header for a document or a section. <footer>: Defines a footer for a document or a section. Meta Information: <meta>: Provides metadata about the HTML document. Miscellaneous: <div>: Defines a division or a section. <span>: Defines an inline container. <br>: Represents a line break. <hr>: Represents a horizontal rule **1.
